기능 요구사항

Schedule

팀원 스케쥴 관리

휴가, 근무, 출장, 교육 등 팀 일정을 캘린더에서 공유합니다. MVP는 일정 CRUD, 캘린더 조회, 유형별 필터를 포함합니다.

Excel

엑셀 업무 자동화

업로드한 엑셀을 특정 컬럼 기준으로 분리하거나 여러 파일을 병합합니다. 결과 파일 다운로드와 처리 이력을 제공합니다.

Chatbot

민원 대응 챗봇

첨부된 민원 매뉴얼을 근거로 대응 방식과 응대 스크립트 초안을 생성합니다. 근거 부족 시 담당자 검토를 안내합니다.

News

뉴스 기사 수집

공공 행정 관련 뉴스를 키워드 기반으로 수집하고 중복 제거 후 날짜와 키워드별 목록을 제공합니다.

운영 가이드

uv 캐시 설정

현재 환경에서는 기본 uv 캐시 경로 권한 문제가 있어 아래 설정을 권장합니다.

$env:UV_CACHE_DIR='C:\Users\admin\Desktop\Day3\.uv-cache'

예상 실행 명령

영역 명령
Frontend npm run dev
Backend uv run uvicorn app.main:app --reload

현재는 실제 앱 코드가 아직 없으므로 위 명령은 후속 구현 이후 사용합니다.

주요 리스크

  • 민원 대응 챗봇 답변은 최종 판단이 아닌 담당자 검토용 초안으로 제한해야 합니다.
  • 엑셀과 민원 매뉴얼에는 개인정보가 포함될 수 있으므로 보관 기간과 접근 권한이 필요합니다.
  • 뉴스 수집은 언론사 약관, 포털 정책, 저작권 범위를 검토해야 합니다.
  • SQLite는 초기 개발에 적합하지만 동시 쓰기 부하가 커지면 DB 전환이 필요합니다.
